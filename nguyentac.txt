Những **nguyên tắc cơ bản khi pianist đánh số ngón tay (piano fingering principles)** thực chất xoay quanh 5 mục tiêu: **tự nhiên – mượt – chuẩn bị trước – ổn định – hiệu quả cơ học**. Dưới đây là các nguyên tắc cốt lõi mà hầu hết pianist và editor (Henle, Alfred, ABRSM…) đều tuân theo.

---

# 1. Ưu tiên vị trí tay tự nhiên (Natural Hand Position)

Bàn tay tự nhiên nhất khi bao phủ **5 phím liên tiếp**.

Ví dụ:

```
C D E F G
1 2 3 4 5
```

Nguyên tắc:

* cố gắng giữ tay trong **five-finger position**
* hạn chế dịch chuyển tay liên tục

Sai:

```
1 3 1 3 1
```

Đúng:

```
1 2 3 4 5
```

---

# 2. Giảm di chuyển tay (Minimum Hand Movement)

Pianist luôn cố:

* giảm **hand shift**
* giảm **nhảy vị trí**

Ví dụ:

Sai:

```
C  D  E  F
1  2  1  2
```

Đúng:

```
C  D  E  F
1  2  3  4
```

Vì bàn tay không cần di chuyển.

---

# 3. Ngón cái là điểm xoay (Thumb Pivot)

Ngón **1 (thumb)** thường dùng để:

* xoay bàn tay
* chuyển position
* chạy scale

Ví dụ scale chuẩn:

Tay phải

```
1 2 3 1 2 3 4 5
```

Tay trái

```
5 4 3 2 1 3 2 1
```

Nguyên tắc:

> luôn lên kế hoạch **thumb crossing**.

---

# 4. Ngón mạnh cho nốt quan trọng

Độ mạnh ngón:

```
1 > 2 > 3 > 4 > 5
```

Do đó:

* melody note → ngón **1,2,3**
* accent → ngón **1 hoặc 2**
* cadence → ngón **3 hoặc 5**

Ví dụ phrase ending:

```
G (whole note)
3
```

thay vì

```
5
```

để kiểm soát tốt hơn.

---

# 5. Tránh ngón 4 nếu có thể

Ngón **4 yếu và ít độc lập**.

Nhiều fingering cố tránh pattern:

```
3 4 5
```

nếu có lựa chọn:

```
2 3 5
```

---

# 6. Tối ưu cho legato

Fingering phải cho phép **nối nốt mượt**.

Ví dụ:

Sai:

```
C  D  E
1  1  1
```

Đúng:

```
C  D  E
1  2  3
```

Vì mỗi ngón có thể giữ nốt trước khi nhấc.

---

# 7. Chuẩn bị cho nốt tương lai (Look-ahead)

Pianist luôn nhìn trước vài nốt.

Ví dụ:

```
C D E F G A B C
```

Nếu nhìn từng nốt:

```
1 2 3 4 5 ?
```

sẽ kẹt ngón.

Nhưng nhìn trước:

```
1 2 3 1 2 3 4 5
```

---

# 8. Fingering theo pattern âm nhạc

Pianist không nghĩ từng nốt mà nghĩ **pattern**.

### Scale

```
1 2 3 1 2 3 4 5
```

### Arpeggio

```
1 2 3 5
```

### Broken chord

```
1 3 5 3
```

### Alberti bass

```
5 2 3 2
```

---

# 9. Reset ở đầu phrase

Nhiều fingering được chọn để **bắt đầu phrase mới ở vị trí tốt**.

Ví dụ:

```
| phrase end | phrase start |

G A B C
2 3 4 5
```

thay vì

```
1 2 3 4
```

để chuẩn bị phrase mới.

---

# 10. Fingering phải cân bằng bàn tay

Tránh:

* kéo ngón quá xa
* xoắn cổ tay

Ví dụ:

Sai:

```
1     5
C     G
```

Nếu có thể:

```
1 2 3 5
C E G C
```

---

# 11. Ưu tiên ngón dài cho phím đen

Nguyên tắc piano phổ biến:

* phím đen → ngón **2,3,4**
* phím trắng → **1,5**

Ví dụ:

```
F# G#
2  3
```

Vì phím đen nằm sâu hơn.

---

# 12. Giữ shape bàn tay

Pianist thường giữ **shape chord**.

Ví dụ C major:

```
C E G
1 3 5
```

nếu lặp lại:

```
1 3 5
```

không đổi.

---

# 13. Nguyên tắc quan trọng nhất

Trong thực tế:

> **Fingering tối ưu cho phrase chứ không phải từng nốt.**

Tức là pianist tối ưu:

```
melodic phrase
hand position
pattern
```

không phải

```
note → finger
```

---

💡 Với hệ thống AI cho piano (như bạn đang xây), các rule cốt lõi nên encode là:

```
1. minimize hand shift
2. prefer scale/arpeggio patterns
3. plan thumb crossing
4. strong finger on strong beat
5. phrase-aware fingering
```
